# FIT 百度React组件库

[fit.baidu.com](http://fit.baidu.com) 

# 特性

- [x] react + redux + fis3&webpack
- [x] typescript源码
- [x] sass模块化
- [x] 支持后端渲染

# 安装

````bash
$ npm install fit-[components]
````

# 用例

````jsx
import Timeago from 'fit-timeago'
ReactDOM.render(<Timeago date={new Date()}/>, reactNode)
````

样式是内置的,无需额外引用
